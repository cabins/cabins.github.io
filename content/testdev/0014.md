## 无处不在的搜索
对于测试开发工程师来说：
    - 合理，并善用搜索，绝对能非常高的提高效率
    - 文件，应用，设置项太多，记忆不是我们的长处，也不是我们该做的事情
    - 快速的找到所需的文件，应用，配置，是追求效率的一种体现

推荐软件：
- macOS
    - SpotLight
    - Alfred
- Windows
    - Listary+Everything
    - Wox+Everything
- Linux
    - Alt+F2
    - Find / Grep / Locate
    - 补充：Albert
- 跨平台
    - utools（Linux仅提供deb包）
    - Cerebro

## 包管理器的使用
对于测试开发工程师来说，包管理器是快速搭建开发环境的一个技巧，也是安装管理应用最便捷的手段。

推荐软件：
- Linux，不同的分发版自带不同的包管理器
    - Ubuntu/Debian, apt
    - Fedora/CentOS/RHEL, yum/dnf
    - ArchLinux, pacman
    - Solus, eopkg
- macOS
    - Homebrew
- Windows
    - Chocolatey
    - Scoop（强烈推荐）
    - Powershell带的包管理器

## Windows上补全Linux命令
对于测试开发工程师来说，弥补Linux命令到其他的平台上，是跨平台高效开发的一种保障。

对于Windows来说，可以用两种方式来补全：
    - 使用Windows Subsystem Linux（WSL），缺点：只适用于Windows 10
    - 使用msys2，Cygwin等方式（建议使用msys2）
