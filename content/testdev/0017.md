## 推荐使用Go Module
```bash
export GO111MODULE=on
```

推荐写入到环境变量中。如：
```bash
echo 'export GO111MODULE=on' >> ~/.bashrc
```

## 使用阿里云代理
```bash
export GOPROXY=https://mirrors.aliyun.com/goproxy/
```

推荐写入到环境变量中。

## 使用 goproxy.cn
```bash
export GOPROXY=https://goproxy.cn
```

推荐写入到环境变量中。
