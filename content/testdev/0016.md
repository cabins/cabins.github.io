## 使用国内镜像站点进行安装
- 使用阿里云镜像
- 使用豆瓣镜像

## 通过-i参数指定
```bash
# 阿里云镜像加速
pip install -i https://mirrors.aliyun.com/pypi/simple/
```

```bash
# 豆瓣镜像加速
pip install -i https://pypi.douban.com/simple/
```

## 通过配置文件配置
在`~/.pip/pip.conf`文件中写入内容

```bash
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/ 

[install]
trusted-host=mirrors.aliyun.com
```
