## Git

- Git 是一个版本控制软件
- Linux 之父——`林纳斯·托瓦兹`为了更好的管理Linux内核代码而编写的
- 分布式版本控制软件
- Github/BitBucket/Gitlab

## 为什么要学习和使用Git

- 可离线工作
- 代码管理
- 优异的分支管理
- 行业标准
- 程序员标配技能
